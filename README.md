## AI-Variable-Name
개발자들을 위한, 변수를 작명해주는 간단한 인공지능 채팅 서비스입니다.
<br>
ChatGpt의 API를 이용하여 프로젝트를 진행하였으며, 
<br>
반응형 디자인으로 제작하였습니다.
